 Eniac Discount Analysis

 Project Overview
This project analyzes how different discount strategies affect sales and revenue using Eniac sales data.

 Objectives
- Analyze the impact of discounts on revenue
- Identify which products benefit most from discounts
- Support data-driven pricing decisions

 Tools & Skills
- Python
- pandas
- matplotlib / seaborn
- Exploratory Data Analysis (EDA)

 Key Insights
- Discounts increase sales volume but reduce profit margins
- Not all products benefit equally from discounts
